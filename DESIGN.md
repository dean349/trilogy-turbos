# Trilogy Turbos - Vibe Coding Playbook

This document defines the strict UI and UX standards (the "Playbook") for all local Vibe Coding generation for Trilogy Turbos LLC. This design system was extracted directly from the authentic MD3 Dark Mode Home Page generated by Google Stitch.

## 1. Core Architecture
- **Framework:** Pure HTML5 + Tailwind CSS (via CDN script).
- **Icons:** Google Material Symbols Outlined.
- **Fonts:** Archivo Narrow (Headlines), Inter (Body), JetBrains Mono (Labels).
- **Theme Engine:** `darkMode: "class"` is HARDCODED on all `<html>` tags. The site runs in persistent dark mode.

## 3. UI/UX Elements & Patterns

### Base Body Classes
`<body class="bg-surface font-body-md text-on-surface antialiased overflow-x-hidden pt-[100px]">`

### Standard Section Container
```html
<section class="py-[80px] bg-surface-container-lowest border-y border-metallic">
    <div class="max-w-container-max mx-auto px-margin-mobile md:px-margin-desktop">
        <!-- Content here -->
    </div>
</section>
```

### Card Component ("Three Pillars" Style)
Used for services, parts, and vehicles.
```html
<div class="border-metallic border rounded-xl overflow-hidden card-glow transition-all duration-300 flex flex-col group bg-surface-container-lowest">
    <div class="h-48 bg-cover bg-center w-full border-b border-metallic" style="background-image: url('URL_HERE');"></div>
    <div class="p-8 flex-grow flex flex-col">
        <div class="flex items-center gap-3 mb-4">
            <span class="material-symbols-outlined text-primary text-[28px]">icon_name</span>
            <h3 class="font-headline-md text-3xl text-on-surface uppercase font-bold">Card Title</h3>
        </div>
        <p class="font-body-md mb-8 flex-grow text-on-surface-variant">Description text here.</p>
        <a class="font-label-caps text-sm text-primary uppercase flex items-center gap-2 group-hover:gap-3 transition-all" href="#">
            Action Link <span class="material-symbols-outlined text-[16px]">arrow_forward</span>
        </a>
    </div>
</div>
```

### Nav & Footer
Every page MUST re-use the exact TopNavBar and Footer code extracted from `TrilogyTurbosHome.html`. The Logo must be the `Trilogy Turbos - modern logo var1 (Electric Blue) - Transparent-.png - website 512 px x 288 px.png` (using the provided lh3.googleusercontent.com CDN link).

### Forms & Inputs (For Contact / Booking)
Use `bg-surface-container border border-metallic text-on-surface rounded-lg p-3 w-full focus:outline-none focus:border-primary transition-colors`.

## 4. E-Commerce Elements (Parts & Vehicles)
- Buttons for Cart/Checkout/Reserve should use `bg-primary-container text-white px-6 py-3 rounded uppercase font-label-caps font-bold hover:opacity-90 transition-opacity`.
- Grids for vehicles/parts should use `grid-cols-1 md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4 gap-gutter`.
